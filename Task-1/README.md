# Portfolio Starter — Rayidi Kavya Chowdary

This repo contains a minimal static portfolio starter implementing the requested dark modern UI, purple/blue gradient, glassmorphism cards, neon hover, animated skill counters, project filtering, resume download, GitHub stats widget, light/dark mode, and scroll progress.

Files added:
- `index.html` — main static page
- `styles.css` — theme and layout
- `scripts.js` — interactivity (theme, counters, filtering, GitHub stat fetch)
- `assets/resume.pdf` — placeholder resume file

Quick start: open `index.html` in a browser. For local development with live reload, use any static server such as `live-server` or VS Code Live Preview.

Notes:
- Replace the username in `scripts.js` `fetchGitHubStats('rayidikavya')` with your GitHub handle.
- Replace `assets/resume.pdf` with your actual PDF resume.
- The starter uses CDN libs: GSAP, AOS, Typed.js, Particles.js, Font Awesome.
