# Task-4: Online Bookstore (Minimal Full-Stack App)

This repository contains a minimal full-stack web application (Online Bookstore) implementing:
- Authentication (register, login) with JWT
- Products CRUD (admin only for create/update/delete)
- Orders creation
- Admin panel with simple stats

Quick start (Windows):

1. Install Node.js (v18+ recommended).
2. Open a terminal in this folder and run:

```powershell
npm install
npm start
```

3. Open http://localhost:3000 in your browser.

Default seeded admin:
- email: `admin@local`
- password: `adminpass`
